A. Data Sources & Acquisition

A.1 IIE Open Doors

The primary enrollment data comes from the Institute of International Education's annual Open Doors report, which has surveyed U.S. institutions on international student enrollment since 1954. Three tables were used:

TableGranularityYears UsedKey Variable
All Places of OriginCountry × Year2000/01–2024/25Korean student national total
Leading InstitutionsInstitution × Year2000/01–2024/25Total international enrollment per institution (top 25 per year); used as a proxy for institutional attractiveness to Korean students
Fields of StudyCountry × Field × Year2009/10–2024/25Korean enrollment share (%) by major category
⚠️ Data note: The IIE "Leading Institutions" table reports total international enrollment at each institution, not Korean-specific counts. Korean-student counts per institution are not published at the institution level by IIE Open Doors. This project uses total international enrollment as an institutional attractiveness proxy. Access to full historical tables may require a direct email request to opendoors@iie.org.

A.2 IPEDS

Institutional-level characteristics (Carnegie classification, tuition, state, control) were drawn from the NCES Integrated Postsecondary Education Data System (IPEDS). Two survey components were used: HD (Institutional Characteristics) and EF (Fall Enrollment). Data is free to download at nces.ed.gov/ipeds/use-the-data with no registration required. The most recent available year of institutional characteristics was used as a time-invariant covariate (treating institution type as stable).

A.3 DHS SEVIS by the Numbers

SEVIS annual PDF reports (2015–2024) provide visa-type and field-of-study breakdowns by nationality that are not available in IIE data. Extracted via tabula-py. Coverage begins only in 2015; earlier years rely solely on IIE for field-of-study data.

A.4 QS World University Rankings

QS rank was used as a proxy for international academic prestige. Official QS data is proprietary; this project uses the CSV edition available on Kaggle (akashbommidi/2026-qs-world-university-rankings), covering 2017–2026. For years before 2017, the earliest available QS rank was carried backward (time-invariant assumption). Rank was transformed as log(rank) to reduce skew before use in regression.

A.5 U.S. Census ACS

Korean-American population by state was obtained from the American Community Survey 5-year estimates (Table B02015). Retrieved via the Census API using the censusdatadownloader Python library. Used as a geographic predictor variable.

A.6 KEDI Korean Outbound Statistics

Korean Ministry of Education outbound student counts were obtained from the Korean Education Development Institute (KEDI) at kess.kedi.re.kr. Used to compute the U.S. share of total Korean outbound students — a more meaningful denominator than raw counts, since it controls for overall changes in Korean study-abroad propensity.

B. Data Cleaning & Integration

B.1 Year Standardization

IIE reports academic years in the format YYYY/YY (e.g., "2020/21"). All year fields were converted to the starting calendar year (integer) using: 

df['year'] = df['academic_year'].str.extract(r'(\d{4})').astype(int)

B.2 Institution Name Standardization

IIE institution names and IPEDS institution names use inconsistent abbreviations and punctuation. A fuzzy matching approach was used to build a crosswalk table against the full IPEDS HD2023 universe (6,163 institutions), followed by manual review and override of all problematic matches: 

from rapidfuzz import process, fuzz

# Normalize names before matching
def normalize_name(name):
    name = str(name).strip()
    name = name.replace('Univ.', 'University').replace('Tech.', 'Technology')
    name = name.replace(' - ', ' ')
    return name

# Match each IIE name against full IPEDS universe
for name in iie_names:
    match, score, _ = process.extractOne(
        name, ipeds_names, scorer=fuzz.token_sort_ratio
    )

Of the 45 IIE Leading Institutions, 45 were matched to IPEDS UNITIDs. Approximately 25 matches required manual override due to common failure patterns: branch campus disambiguation (e.g., "Indiana University Bloomington" fuzzy-matched to "Dominican University"), system vs. flagship confusion (Texas A&M → Georgia College), and UC campus ambiguity (UCLA, UCSD, UCB share similar name stems). A 45-entry manual override dictionary was built using UNITID values verified directly from HD2023.

⚠️ IIE "Leading Institutions" data reports only the top 25 universities per sending country per year. Institutions outside the top 25 in any given year have their Korean enrollment counts suppressed. This creates a censored outcome problem: the dataset understates enrollment at non-top-25 institutions. All analyses were conducted on the observed top-25 data with this limitation noted.

C. Analytical Methods

C.1 Trend Analysis

Total Korean enrollment over time was modeled using a piecewise linear regression to identify structural breaks, with break points corresponding to known policy events (2016 STEM OPT expansion, 2020 COVID onset). The Chow test was applied to confirm statistical significance of breaks.

C.2 Correlation Analysis

Spearman rank correlations were computed between Korean enrollment and each predictor variable. Spearman was chosen over Pearson because QS rank, Korean enrollment, and Korean community size are all right-skewed. For each bivariate relationship, a scatterplot was produced (see interactive figure I2). ★ = statistically significant at α = 0.05.

C.3 Comparative Trend (Korea vs. China vs. India)

To contextualize Korean trends, Chinese and Indian enrollment series were extracted from the same IIE "All Places of Origin" dataset and normalized to an index (base year = 2000, set to 100). This reveals divergence in growth trajectories without being confounded by raw population differences.

C.4 Field-of-Study Shift

Major categories were aggregated from IIE's detailed field codes into four groups: Business, STEM, Humanities & Social Sciences, and Other. Share of total Korean enrollment was computed per group per year. A before/after comparison around the 2016 STEM OPT expansion was performed using a difference-in-differences framework comparing STEM vs. non-STEM enrollment trends.

D. Visualization Technical Notes

D.1 Color Palette

All visualizations use the project palette defined in website/css/theme.css. The palette was tested for color-blind accessibility using the Coblis simulator. Primary data series use navy (#1A3A5C); the Korean accent color (#C0392B) is used sparingly for annotations and highlights.

D.2 Static Figures (matplotlib)

Generated via notebooks/03_static_viz.ipynb. Exported at 300 DPI as PNG. Font: Inter (sans-serif) for axis labels; Georgia (serif) for titles. All axis labels and titles are set explicitly; no default matplotlib style was used.

D.3 Interactive Visualizations (Plotly / D3.js)

All interactive figures are self-contained HTML files with no external dependencies beyond CDN-hosted libraries. They require no backend server and are safe to host on GitHub Pages. The choropleth map uses GeoJSON boundary data from the U.S. Census Cartographic Boundary files (20m resolution, 2022 vintage).

D.4 Linked View (D3.js)

The field-of-study linked view was built in D3.v7. The donut chart and line chart share a single data object in memory. Click events on the donut's arc segments call a shared updateLineChart(field) function that filters the pre-loaded dataset and re-renders the line with a 300ms transition.

E. Limitations & Caveats

  • IIE "Leading Institutions" reports only the top 25 universities per country per year — enrollment at other institutions is not captured.
  • QS rankings were not available before 2004; earlier years use the first available rank as a proxy.
  • STEM OPT eligibility is based on program-level CIP codes, not student-level outcomes; actual OPT take-up rates are not available per institution.
  • KEDI outbound data includes all destinations; any changes in Korean study-abroad policy could confound the U.S.-share denominator.
  • Tuition figures are list prices, not net prices — Korean students may pay very different amounts based on scholarships and aid.
  • All correlations are observational; no causal claims are made.
